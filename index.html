Clinic Operations Optimization with Python

Improving clinic resource planning through data science and project management

Project Overview

This project uses Python, data science, and project management principles to optimize clinic operations such as staff scheduling and patient flow.

  • Tools Used: Python (Pandas, NumPy, Matplotlib), Project Management, Public Health Knowledge
  • Focus: Simulating clinic patient flow and resource usage to improve efficiency
  • Outcome: Developed a model for scheduling optimization and resource allocation
